# Pokedex

Este projeto é uma Pokédex web desenvolvida em HTML, CSS e JavaScript, que consome a API pública do PokéAPI para exibir informações dos Pokémons.

## Demonstração
- Pesquise pelo nome ou número do Pokémon.
- Navegue entre os Pokémons usando os botões de próximo e anterior.
- Layout  com vídeo de fundo animado.

## Como rodar
1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
